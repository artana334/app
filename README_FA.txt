این پروژه یک پروژه کامل Gradle برای Android Studio است.

روش باز کردن:
Android Studio -> File -> Open -> پوشه PetroArtanExhibitionForm (پوشه ریشه) را انتخاب کنید.

لوگو:
لوگوی شرکت را با نام logo.png در app/src/main/res/drawable/ قرار دهید
سپس در activity_form.xml مقدار android:src را از @drawable/logo_placeholder به @drawable/logo تغییر دهید.

SMTP:
به خاطر امنیت، پسورد داخل کد نیست.
بعداً می‌توانیم پسورد را از BuildConfig یا local.properties بخوانیم.
